# Wikidata Query Builder

## Demo
You can find a demo version on https://tools.wmflabs.org/wd-query-builder/.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
